# i-BrainTech landing page

Intended URL path: /for/i-brain-tech

A single-page site built for Dr. Konstantin Sonkin and the i-BrainTech team ahead of eMerge Americas 2026 (Booth 525). The page carries the 60/30/10 read on i-BrainTech's operational stack, a Feeld-shape comparator sprint, an Ethics Engine governance reference, the NLP Logix partnership line for work below the orchestration layer, and a direct CTA to Matt Creamer's calendar.

## Files

- `index.html` — the page
- `styles.css` — all styles, no JavaScript
- `logo.svg`, `logo-white.svg` — i-BrainTech marks used in the top bar

## Brand alignment

- Palette pulled from the i-brain.tech Webflow `:root`: space-blue `#123358`, turquoise `#00a8ac`, cinnabar `#d64933`.
- Typography is Raleway across the page, matching the live site body family. The proprietary "Rebrand" webfont is not publicly hosted, so headings use Raleway 500 per the brand-pack substitution note.
- Dark navy hero with a cinnabar CTA echoes the live hero pattern.

## Voice and disclosure

- Copy follows the voice rules: no em dashes, no antithesis patterns, no performed questions, no hype words, no pricing.
- Ethics Engine is cited with a one-sentence bio and real anchor links to arxiv and GitHub.
- NLP Logix reference carries the mandatory credibility line ("in machine learning since 2011" and "over 150 data scientists") on one line.
- Eduba branding appears only in the footer line "Prepared by Eduba, eduba.io" per the shared chrome rules.
